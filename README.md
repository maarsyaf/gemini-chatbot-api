# Narabot - Simple Chatbot with powered by Google Gemini AI

## Cara pakai (singkat)
1. Clone:
```bash
git clone https://github.com/maarsyaf/gemini-chatbot-api.git
cd gemini-chatbot-api
```
2. Buat `.env`:
```
API_KEY=your_gemini_api_key_here
PORT=3000
```
4. Jalankan:
```bash
node index.js
nodemon index.js

```
API berjalan di `http://localhost:3000`.

## Tujuan
Submission Hacktiv8: mengintegrasikan Gemini AI dengan simple chatbot yang responsif.

## Tech
- Node.js, Express  
- Cors  
- dotenv  
- Gemini 2.5 Flash
- HTML, CSS, Javascript (Simple Website)
